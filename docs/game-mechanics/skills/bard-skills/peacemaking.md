# Peacemaking

## Overview

Peacemaking, if successful, will stop all nearby creatures in place and prevent them from fighting for a very short period.

To use Peacemaking, you need instruments and Musicianship to play them.

## Training

Peacemaking can be used on a 10 seconds cooldown, it doesn't have any difficulty checks. Just use it until you reach 100.

To be more efficient, you can gain Peacemaking and Musicianship at the same time.
